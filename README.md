# Studegram - Student Application Portal

A modern B2B recruitment agent portal for managing international student university applications.

[Add your Vercel URL here]

## 📋 Features
- **Dashboard** — Application stats, important notices, university deadlines
- **Application History** — Manage and track all student applications
- **Search Courses** — Browse 100,000+ courses across universities
- **Notice Board** — Stay updated with scholarship and intake alerts
- **University Deadlines** — Track application and scholarship deadlines
- **Add New Application** — 3-step modal flow (Select Course → Eligibility Check → Student Form)
- **Login & Registration** — Agent authentication pages

## 🛠️ Tech Stack
- React 18
- Vite
- Tailwind CSS
- JavaScript (ES6+)

## 📦 Installation

```bash
git clone https://github.com/YOUR_USERNAME/studegram-portal.git
cd studegram-portal
npm install
npm run dev
```
